ePortfolio Generator

December 2015

A project built using the JavaFX library. It is mostly front-end heavy. Users can make multiple web pages and add elements such as images, videos, paragraphs and slideshows. It currently comes with 5 prexisting layouts (made from scratch), 5 color schemes, and 5 font choices. Users can save their work as a JSON file and open it up later. Used BitBucket for version control but imported the repository to GitHub.



Client Art Portfolio

December 2016

A website for a client's art portfolio. I used Bootstrap to make sure it was mobile friendly. I also designed the logo using GIMP. The logo in addition to the about picture uses webkit animations to add something a little more artsy.

I used Salvatorre to make it have a blog-esque look and LightBox to display a slideshow for the images.
